Connect the WhatsApp Business API to send campaign outreach, bulk messages, and supporter notifications via WhatsApp. Requires a Meta Business account.
Required Fields
Connector typeWhatsApp Business
Namee.g. Campaign WhatsApp
Providermeta
API Base URLhttps://graph.facebook.com/v18.0
Access TokenThe permanent System User token from your Meta Business account. Not a temporary user token.
Phone Number IDFound in Meta Business Manager → WhatsApp → API Setup. A numeric ID like 1234567890
Business Account IDYour WhatsApp Business Account (WABA) ID from Meta Business Manager.
Default TemplateName of your approved message template, e.g. campaign_greeting
How to get these credentials
  1. Go to business.facebook.com → Business Settings
  2. Under Accounts → WhatsApp Accounts, find your WABA ID
  3. Go to System Users → Add a system user with Admin role
  4. Click Generate Token → select the WhatsApp app → set expiry to Never
  5. In developers.facebook.com → your app → WhatsApp → API Setup → copy the Phone number ID
Use a System User token, not your personal access token. Personal tokens expire and will break messaging.
Webhook (optional)
Webhook URLYour server's callback URL for incoming messages, e.g. https://campaign.webmasta.com.pg/api/webhooks/whatsapp/
Webhook SecretA random string you define. Enter this as the Verify Token in the Meta webhook setup.
Connect an SMS gateway to send bulk campaign messages, GOTV reminders, and supporter alerts via text message. Supports Twilio, Vonage, Africa's Talking, and any provider with an HTTP API.
Required Fields

Namee.g. Twilio SMS or Africa's Talking
Providertwilio / vonage / africastalking / custom
API Base URLTwilio: https://api.twilio.com/2010-04-01
Africa's Talking: https://api.africastalking.com/version1
API KeyYour Account SID (Twilio) or username/API key from the provider
API SecretAuth Token (Twilio) or API secret from the provider
SMS Sender IDYour approved sender name or number, e.g. CAMPAIGN or +67570012345
Default Country Code+675 for Papua New Guinea (pre-filled)
PNG-Specific Notes

PNG mobile numbers are 8 digits after the country code. Always store numbers with +675 prefix. The platform auto-formats numbers on import if you set the default country code correctly.

Digicel PNG and Bmobile are the main carriers. Not all SMS gateways have reliable reach to PNG — test before going live.
Configure an SMTP email server to send transactional emails, reports, and bulk campaign communications from the platform.
Required Fields
Connector typeEmail
Namee.g. SendGrid SMTP or Gmail Relay
Providersendgrid / mailgun / gmail / smtp
Email HostSendGrid: smtp.sendgrid.net
Mailgun: smtp.mailgun.org
Gmail: smtp.gmail.com
Email Port587 for TLS (recommended) · 465 for SSL · 25 for plain
Email UsernameSendGrid: apikey (literal) · Gmail: your full email address
Email PasswordSendGrid: your API key · Gmail: an App Password (not your login password)
Use TLSLeave checked unless using port 25
From AddressThe sender address recipients see, e.g. campaign@webmasta.com.pg
Gmail App Password
  1. Enable 2-Factor Authentication on the Gmail account
  2. Go to myaccount.google.com → Security → App Passwords
  3. Select Mail and Other device → Generate
  4. Copy the 16-character password (no spaces) — paste into Email Password
Do not use your Google account password directly — it will be rejected. You must use an App Password.
Connect a mapping service to display ward maps, village locations, and geo-tagged field data within CampaignMasta. Uses Google Maps or a compatible tile service.
Required Fields
Connector typeMaps / GIS
Namee.g. Google Maps
Providergooglemaps / mapbox / openstreetmap
Maps API KeyYour Google Maps JavaScript API key or Mapbox public token
Google Maps Setup
  1. Go to console.cloud.google.com → APIs & Services → Credentials
  2. Click Create Credentials → API Key
  3. Restrict the key to: Maps JavaScript API and Geocoding API
  4. Under Application restrictions, add your domain: campaign.webmasta.com.pg
Google Maps has a free tier of $200/month credit, which covers most campaign usage. Enable billing to avoid the key being disabled.
